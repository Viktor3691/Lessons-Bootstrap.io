Bootstrap 5 Начало работы

Что такое Bootstrap?

  • Bootstrap — это бесплатный интерфейсный фреймворк для более быстрой и простой веб-разработки
  • Bootstrap включает в себя шаблоны дизайна на основе HTML и CSS для типографики, форм, кнопок, таблиц, навигации, модальных окон, каруселей изображений и многого другого, а также дополнительные плагины JavaScript
  • Bootstrap также позволяет легко создавать адаптивные дизайны

Изучение Bootstrap

Bootstrap – это не просто набор стилей, а целая экосистема, упрощающая разработку адаптивных и отзывчивых веб-сайтов. Его популярность обусловлена скоростью разработки, последовательностью дизайна и кросс-браузерной совместимостью.


Основы сетки:

Сетка Bootstrap является краеугольным камнем структуры любого сайта. Она базируется на 12-колоночной системе, позволяющей гибко располагать контент. Понимание классов container, row и col-*-* – ключ к управлению макетом. container центрирует содержимое на странице, row создает горизонтальные полосы, а col-*-* определяет ширину колонки на разных устройствах.

Компоненты Bootstrap:

Bootstrap предоставляет широкий спектр готовых компонентов: навигационные панели, кнопки, формы, модальные окна, карусели и многое другое. Использование этих компонентов значительно ускоряет разработку, так как избавляет от необходимости писать код с нуля. Каждый компонент имеет свои CSS-классы и JavaScript-функции, облегчающие его настройку и интеграцию.

Адаптивность и отзывчивость:

Адаптивность Bootstrap обеспечивается медиа-запросами, которые позволяют применять разные стили в зависимости от размера экрана устройства. Классы, такие как col-sm-*, col-md-*, col-lg-*, и col-xl-*, определяют ширину колонки на маленьких, средних, больших и очень больших экранах, соответственно. Это позволяет создавать сайты, которые выглядят отлично на любых устройствах.
Кастомизация Bootstrap:
Хотя Bootstrap предлагает стили «из коробки», часто требуется кастомизация для соответствия уникальному дизайну проекта. Это можно сделать несколькими способами: переопределение стилей в собственном CSS-файле, использование SASS для изменения переменных Bootstrap, или создание пользовательской сборки Bootstrap, включающей только необходимые компоненты.

Версии Bootstrap

Bootstrap 5 (выпущен в 2021 году) — это новейшая версия Bootstrap (выпущенной в 2013 году) с новыми компонентами, более быстрой загрузкой стилей и улучшенной адаптивностью.

Bootstrap 5 поддерживает последние стабильные версии всех основных браузеров и платформ. Однако Internet Explorer 11 и более ранние версии не поддерживаются.

Основное отличие Bootstrap 5 от Bootstrap 3 и 4 заключается в том, что Bootstrap 5 перешёл на ванильный JavaScript вместо jQuery.

Bootstrap 3 Примечание: и Bootstrap 4 по-прежнему поддерживаются командой разработчиков для исправления критических ошибок и внесения изменений в документацию. Их использование совершенно безопасно. Однако новые функции в них добавляться не будут.

Зачем использовать Bootstrap?

  • Простота в использовании: Любой, кто обладает базовыми знаниями HTML и CSS, может начать использовать Bootstrap
  • Адаптивные функции: Адаптивный CSS в Bootstrap подстраивается под телефоны, планшеты и настольные компьютеры
  • Подход, ориентированный на мобильные устройства: В Bootstrap стили, ориентированные на мобильные устройства, являются частью основного фреймворка
  • Совместимость с браузерами: Bootstrap 5 совместим со всеми современными браузерами (Chrome, Firefox, Edge, Safari и Opera). Обратите внимание, что если вам нужна поддержка IE11 и более ранних версий, вам нужно использовать либо BS4, либо BS3.

Две базовые страницы Bootstrap по 5 страниц

В следующем примере показан код для базовой страницы Bootstrap 5 (с адаптивным контейнером фиксированной ширины).:

Container Example

<!DOCTYPE html>
<html lang="en">
<head>
  <title>Bootstrap Example</title>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
  <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
</head>
<body>

<div class="container">
  <h1>My First Bootstrap Page</h1>
  <p>This part is inside a .container class.</p>
  <p>The .container class provides a responsive fixed width container.</p>
</div>

</body>
</html>

В следующем примере показан код для базовой страницы Bootstrap 5 (с контейнером полной ширины).:

Container Fluid Example

<!DOCTYPE html>
<html lang="en">
<head>
  <title>Bootstrap Example</title>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">

  <p>This part is inside a .container-fluid class.</p>

</div>

</body>
</html>
