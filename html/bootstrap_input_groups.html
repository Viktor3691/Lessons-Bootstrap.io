Группы ввода

Класс .input-group — это контейнер для улучшения ввода путем добавления значка, текста или кнопки перед или за полем ввода в качестве «текста справки».

Чтобы стилизовать указанный текст справки, используйте класс .input-group-text:

Пример

<div class="input-group mb-3">
    <span class="input-group-text">@</span>
    <input type="text" class="form-control" placeholder="Username" style="z-index:0">
</div>
@
end.

Пример

<div class="input-group mb-3">
    <input type="text" class="form-control" placeholder="Your Email" style="z-index:0">
    <span class="input-group-text">@example.com</span>
</div>
@example.com
end.

Размер входной группы

Используйте класс .input-group-sm для небольших групп ввода и .input-group-lg для больших. группы входов:

Пример

<div class="input-group mb-3 input-group-sm">
    <span class="input-group-text">Small</span>
    <input type="text" class="form-control" style="z-index:0">
</div>
Small
end.

Пример

<div class="input-group mb-3">
    <span class="input-group-text">Default</span>
    <input type="text" class="form-control" style="z-index:0">
</div>
Default
end.

Пример

<div class="input-group mb-3 input-group-lg">
    <span class="input-group-text">Large</span>
    <input type="text" class="form-control" style="z-index:0">
</div>
Large
end.

Несколько входов и помощников

Добавьте несколько входов или дополнений:

Пример

<div class="input-group mb-3">
    <span class="input-group-text">Person</span>
    <input type="text" class="form-control" placeholder="First Name" style="z-index:0">
    <input type="text" class="form-control" placeholder="Last Name" style="z-index:0">
</div>
Person
end.

Пример

<div class="input-group mb-3">
    <span class="input-group-text">One</span>
    <span class="input-group-text">Two</span>
    <span class="input-group-text">Three</span>
    <input type="text" class="form-control" style="z-index:0">
</div>
One Two Three
end.

Группа ввода с флажками и радио

Вместо текста вы также можете использовать флажки или переключатели:

Пример

<div class="input-group mb-3">
    <div class="input-group-text">
        <input type="checkbox" style="z-index:0">
    </div>
    <input type="text" class="form-control" placeholder="Some text" style="z-index:0">
</div>
end.

Пример

<div class="input-group mb-3">
    <div class="input-group-text">
        <input type="radio" style="z-index:0">
    </div>
    <input type="text" class="form-control" placeholder="Some text" style="z-index:0">
</div>
end.

Кнопки группы ввода

Пример

<div class="input-group mb-3">
    <button class="btn btn-outline-primary" type="button" style="z-index:1">Basic Button</button>
    <input type="text" class="form-control" placeholder="Some text" style="z-index:0">
</div>
end.

Пример

<div class="input-group mb-3">
    <input type="text" class="form-control" placeholder="Search" style="z-index:0">
    <button class="btn btn-success" type="button" style="z-index:1">Go</button>
</div>
end.

Пример

<div class="input-group mb-3">
    <input type="text" class="form-control" placeholder="Something clever.." style="z-index:0">
    <button class="btn btn-primary" type="button" style="z-index:1">OK</button>
    <button class="btn btn-danger" type="button" style="z-index:1">Cancel</button>
</div>
end.

Группа ввода с раскрывающимся списком

Добавьте раскрывающуюся кнопку в группу ввода. Обратите внимание, что вам не нужна оболочка .dropdown, как обычно.

Пример

<div class="input-group mt-3 mb-3" style="z-index:1">
    <button type="button" class="btn btn-primary dropdown-toggle" data-bs-toggle="dropdown">
        Dropdown button
    </button>
    <ul class="dropdown-menu" style="z-index:0">
        <li><a class="dropdown-item" href="javascrip:void(0)">Link 1</a></li>
        <li><a class="dropdown-item" href="javascrip:void(0)">Link 2</a></li>
        <li><a class="dropdown-item" href="javascrip:void(0)">Link 3</a></li>
    </ul>
    <input type="text" class="form-control" placeholder="Username">
</div>
  • Link 1
  • Link 2
  • Link 3
end.
