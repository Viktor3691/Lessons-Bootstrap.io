Базовые панель навигации

Панель навигации Bootstrap 5 — это простая оболочка для размещения фирменного стиля, навигации и других элементов в навигационном заголовке.

Пример

<nav class="navbar navbar-expand-sm navbar-light bg-faded">
    <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#nav-content"
        aria-controls="nav-content" aria-expanded="false" aria-label="Переключатель навигации">
        <span class="navbar-toggler-icon"></span>
    
    <!-- Бренд -->
    <a class="navbar-brand" href="#">Лого</a>
    <!-- Ссылки -->
    <div class="collapse navbar-collapse" id="nav-content">
        <ul class="navbar-nav">
            <li class="nav-item">
                <a class="nav-link" href="#">Ссылка 1</a>
            </li>
            <li class="nav-item">
                <a class="nav-link" href="#">Ссылка 2</a>
            </li>
            <li class="nav-item">
                <a class="nav-link" href="#">Ссылка 3</a>
            </li>
        </ul>
    </div>
</nav>

Вот объяснение:

  • Заключите всё в элемент <nav> с классом .navbar и классом .navbar-expand{-sm|-md|-lg|-xl|-xxl}, а также выберите цветовую схему.
  • Для переключений меню на небольших устройствах используйте элемент <button> с классом .navbar-toggler. Для отображения «гамбургера» используйте класс .navbar-toggler-icon в элементе <span>.
  • Для списка ссылок используйте элемент <ul> с классом .navbar-nav.
  • Для каждого отдельного элемента списка используйте <li> с классом .nav-item.
  • Для отдельных ссылок используйте <a> с классом .nav-link.

Панель навигации с выпадающим меню

Вы можете добавить выпадающие меню в свою панель навигации. Просто вставьте код выпадающего меню в элемент <li> с классом .dropdown.

Пример

<nav class="navbar navbar-expand-sm navbar-light bg-faded">
    <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#nav-content"
        aria-controls="nav-content" aria-expanded="false" aria-label="Переключатель навигации">
        <span class="navbar-toggler-icon></span>
    </button>
    <!-- Бренд -->
    <a class="navbar-brand" href="#">Логотип</a>
    <!-- Ссылки -->
    <div class="collapse navbar-collapse" id="nav-content">
        <ul class="navbar-nav">
            <li class="nav-item">
                <a class="nav-link" href="#">Ссылка 1</a>
            </li>
            <li class="nav-item">
                <a class="nav-link" href="#">Ссылка 2</a>
            </li>
            <li class="nav-item dropdown">
                <a class="nav-link dropdown-toggle" data-bs-toggle="dropdown" id="Preview" href="#" role="button"
                    aria-haspopup="true" aria-expanded="false">
                    Выпадающее меню
                </a>
                <div class="dropdown-menu" aria-labelledby="Просмотр">
                    <a class="dropdown-item" href="#">Действие</a>
                    <a class="dropdown-item" href="#">Другое действие</a>
                    <a class="dropdown-item" href="#">Что-то ещё</a>
                </div>
            </li>
        </ul>
    </div>
</nav>

Текущая страница

Используйте класс .active, чтобы отметить ссылку, ведущую на текущую страницу или раздел (примените его к <li>).

Также добавьте <span> с классом .sr-only, чтобы сообщить об этом экранным читалкам.

Пример

<nav class="navbar navbar-expand-sm navbar-light bg-faded">
    <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#nav-content"
        aria-controls="nav-content" aria-expanded="false" aria-label="Переключатель навигации">
        <span class="navbar-toggler-icon"></span>
    </button>
    <!-- Бренд -->
    <a class="navbar-brand" href="#">Логотип</a>
    <!-- Ссылки -->
    <div class="collapse navbar-collapse" id="nav-content">
        <ul class="navbar-nav">
            <li class="nav-item">
                <a class="nav-link" href="#">Ссылка 1</a>
            </li>
            <li class="nav-item active">
                <a class="nav-link" href="#">Ссылка 2 <span class="sr-only">(текущая)</span></a>
            </li>
            <li class="nav-item">
                <a class="nav-link" href="#">Ссылка 3</a>
            </li>
        </ul>
    </div>
</nav>

Выравнивание

Для выравнивания элементов на панели навигации можно использовать любой из вспомогательных классов, предназначенных для выравнивания (например, .me-auto или .justify-content-end).

Пример

<nav class="navbar navbar-expand-sm navbar-light bg-faded">
    <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#nav-content"
        aria-controls="nav-content" aria-expanded="false" aria-label="Toggle navigation">
        <span class="navbar-toggler-icon"></span>
    </button>
    <!-- Бренд -->
    <a class="navbar-brand" href="#">Логотип</a>
    <!-- Ссылки -->
    <div class="collapse navbar-collapse justify-content-end" id="nav-content">
        <ul class="navbar-nav">
            <li class="nav-item">
                <a class="nav-link" href="#">Ссылка 1</a>
            </li>
            <li class="nav-item">
                <a class="nav-link" href="#">Ссылка 2</a>
            </li>
            <li class="nav-item">
                <a class="nav-link" href="#">Ссылка 3</a>
            </li>
        </ul>
    </div>
</nav>

Флекс-классы берут свои имена из фактического объявления свойств и значений флексов. Это позволяет легко определить имя наиболее подходящего класса для конкретной ситуации.

Вы также можете добавить адаптивные сокращения, чтобы выравнивание действовало только в определённых размерах области просмотра. Например, .justify-content-lg-end применяет justify-content: end на больших устройствах и выше.

Документация Bootstrap содержит полный список флекс-классов, который вы можете использовать в качестве справочника.
