Работа с типографикой в Bootstrap

В Bootstrap 5 по умолчанию используется значение font-size в 1rem (по умолчанию 16 пикселей), а значение line-height равно 1,5.

Кроме того, все <p> элементы имеют значения margin-top: 0 и margin-bottom: 1rem (по умолчанию 16 пикселей).

<h1> — <h6>

Заголовки HTML в Bootstrap 5 (от <h1> до <h6>) с более жирным шрифтом и адаптивным размером шрифта.

Пример

<h1>h1 Bootstrap heading</h1>
<h2>h2 Bootstrap heading</h2>
<h3>h3 Bootstrap heading</h3>
<h4>h4 Bootstrap heading</h4>
<h5>h5 Bootstrap heading</h5>
<h6>h6 Bootstrap heading</h6>

end.

Отображаемые заголовки

Отображаемые заголовки выделяются сильнее, чем обычные (более крупный шрифт и менее насыщенная текстура), и на выбор предлагается шесть классов: .display-1 — .display-6:

Пример

<div class="display-1">.display-1</div>
<div class="display-2">.display-2</div>
<div class="display-3">.display-3</div>
<div class="display-4">.display-4</div>
<div class="display-5">.display-5</div>
<div class="display-6">.display-6</div>

end.

<small>

В Bootstrap 5 элемент HTML <small> (и класс .small ) используется для создания дополнительного текста меньшего размера в любом заголовке.

Пример

<div class="container mt-3">
    <h1>Smaller, Secondary Text</h1>
    <p>The small element (and the .small class) is used to
     create a smaller, secondary text in any heading:</p>
    <h1>h1 heading <small>secondary text</small></h1>
    <h2>h2 heading <small>secondary text</small></h2>
    <h3>h3 heading <small>secondary text</small></h3>
    <h4>h4 heading <small>secondary text</small></h4>
    <h5>h5 heading <small>secondary text</small></h5>
    <h6>h6 heading <small>secondary text</small></h6>
</div>

end.

Используйте элемент mark, чтобы <mark style="background-color:#eff309;padding:.2em;">выделить</mark> текст.

<abbr>

Bootstrap 5 стилизует элемент HTML <abbr> пунктирной нижней границей и курсором со знаком вопроса при наведении:

Пример

<abbr title="Всемирная Организация здравоохранения"
style="cursor: help;">ВОЗ</abbr> была основана в 1948 году.

ВОЗ была основана в 1948 году.
end.

<blockquote>

Добавьте класс .blockquote к <blockquote> при цитировании блоков контента из другого источника. А при указании источника, например «с сайта WWF», используйте класс .blockquote-footer:

Пример

<div class="container-fluid mt-2">
    <blockquote class="blockquote">
     <Вот уже 50 лет WWF защищает будущее природы.
     Ведущая в мире природоохранная организация,
     Всемирный фонд дикой природы работает в 100 странах и поддерживается
     1,2 миллиона членов в Соединенных Штатах и почти 5
     миллионов по всему миру.
        <footer class="blockquote-footer">From WWF's website</footer>
    </blockquote>
</div>

<dl>

Bootstrap 5 будет стилизовать элемент HTML <dl> следующим образом:

Пример

<dl>
  <dt>Coffee</dt>
  <dd>- black hot drink</dd>
  <dt>Milk</dt>
  <dd>- white cold drink</dd>
</dl>
Coffee
- black hot drink
Milk
- white cold drink
end.

<code>

Bootstrap 5 будет стилизовать элемент HTML <code> следующим образом:

Пример

<div class="container mt-3">
    <h1>Code Snippets</h1>
    <p>Inline snippets of code should be embedded  in the code element:</p>
    <p>The following HTML elements: <code>span</code>, <code>section</code>, and <code>div</code>
     defines a section in a  document.</p>
</div>

Code Snippets

Inline snippets of code should e embedded in the code element:

The following HTML elements: span, section, and div defines a section in a document.

end.

<kbd>

Bootstrap 5 будет стилизовать элемент HTML <kbd> следующим образом:

Пример

<div class="container mt-3">
    <h>>Ввод с клавиатуры</h1>
    <p>Чтобы указать вводимые данные, которые обычно вводятся с клавиатуры, используйте элемент kbd:</p>
    <p>Используйте <kbd>ctrl + p</kbd> чтобы открыть диалоговое окно печати.</p>
</div>

Ввод с клавиатуры

Чтобы указать вводимые данные, которые обычно вводятся с клавиатуры, используйте элемент kbd:

Используйте ctrl + p чтобы открыть диалоговое окно печати.

end.

Дополнительные классы типографики

Для дальнейшего стилизации HTML-элементов можно добавить следующие классы Bootstrap 5:

Класс Описание
.lead Выделяет абзац
.text-start Обозначает текст, выровненный по левому краю
.text-break Предотвращает разрыв макета из-за длинного текста
.text-center Обозначает текст, выровненный по центру
.text-decoration-none Убирает подчёркивание со ссылки
.text-end Обозначает текст, выровненный по правому краю
.text-nowrap Указывает на отсутствие текста переноса
.text-lowercase Текст в нижнем регистре
.text-uppercase Обозначает текст в верхнем регистре
.text-capitalize Указывает на текст , написанный заглавными буквами
.initialism Текст внутри элемента <abbr> отображается шрифтом немного меньшего размера
.list-unstyled Удаляет стиль списка по умолчанию и левое поле у элементов списка (работает как с <ul>, так и с <ol>). Этот класс применяется только к непосредственным дочерним элементам списка (чтобы удалить стиль списка по умолчанию из любых вложенных списков, примените этот класс к любым вложенным спискам)
.list-inline Выводит все элементы списка в одну строку (используется вместе с .list-inline-item для каждого элемента <li>)
