Модальные окна

❮ ❮ ❮ Предыдущий
Следующий ❯ ❯ ❯


Bootstrap позволяет добавить на ваш сайт модальное диалоговое окно. Модальное окно — это диалоговое окно, которое получает фокус, в то время как остальная часть экрана затемнена или выделена серым цветом. Это заставляет пользователя выполнить действие в диалоговом окне, прежде чем продолжить.

Чтобы создать модальное окно, используйте класс .modal вместе с другими классами .modal-* для определения каждого раздела модального окна.

Пример

<button type="button" class="btn btn-primary" data-bs-toggle="modal" data-bs-target="#exampleModal">
    Открыть модальное окно
</button>
<div class="modal fade" id="exampleModal" tabindex="-1" aria-labelledby="exampleModalLabel" aria-hidden="true">
    <div class="modal-dialog">
        <div class="modal-content">
            <div class="modal-header">
                <h5 class="modal-title" id="exampleModalLabel">Заголовок</h5>
                <button type="button" class="btn-close" data-bs-dismiss="modal" aria-label="Закрыть"></button>
            </div>
            <div class="modal-body">
                Содержимое…
            </div>
            <div class="modal-footer">
                <button type="button" class="btn btn-secondary" data-bs-dismiss="modal">Закрыть</button>
                <button type="button" class="btn btn-primary">Сохранить изменения</button>
            </div>
        </div>
    </div>
</div>

Статичный фон

Модальное окно в предыдущем примере будет закрываться всякий раз, когда пользователь щёлкает за его пределами. Вы можете убрать эту возможность, сделав следующее:

Пример

<button type="button" class="btn btn-primary" data-bs-toggle="modal" data-bs-target="#staticBackdrop">
    Открыть модальное окно
</button>
<div class="modal fade" id="staticBackdrop" data-bs-backdrop="static" data-bs-keyboard="false" tabindex="-1"
    aria-labelledby="staticBackdropLabel" aria-hidden="true">
    <div class="modal-dialog">
        <div class="modal-content">
            <div class="modal-header">
                <h5 class="modal-title" id="staticBackdropLabel">Заголовок</h5>
                <button type="button" class="btn-close" data-bs-dismiss="modal" aria-label="Закрыть"></button>
            </div>
            <div class="modal-body">
                Содержимое…
            </div>
            <div class="modal-footer">
                <button type="button" class="btn btn-secondary" data-bs-dismiss="modal><Закрыть</button>
                <button type="button" class="btn btn-primary">Понятно</button>
            </div>
        </div>
    </div>
</div>
