Bootstrap 5 Группа кнопок

❮ ❮ ❮ Предыдущий
Следующий ❯ ❯ ❯

Группа кнопок Bootstrap позволяет группировать кнопки по горизонтали или вертикали.

Горизонтальная группа кнопок

Большие кнопки:

Пример

<div class="btn-group btn-group-lg">
    <button type="button" class="btn btn-primary">Apple</button>
    <button type="button" class="btn btn-primary">Samsung</button>
    <button type="button" class="btn btn-primary">Sony</button>
</div>
end.

Кнопки по умолчанию:

Пример

<div class="btn-group">
    <button type="button" class="btn btn-primary">Apple</button>
    <button type="button" class="btn btn-primary">Samsung</button>
    <button type="button" class="btn btn-primary">Sony</button>
</div>
end.

Маленькие кнопки:

Пример

<div class="btn-group btn-group-sm">
    <button type="button" class="btn btn-primary">Apple</button>
    <button type="button" class="btn btn-primary">Samsung</button>
    <button type="button" class="btn btn-primary">Sony</button>
</div>
end.

Группы вертикальных кнопок

Bootstrap 5 также поддерживает группы вертикальных кнопок:

Пример

<div class="btn-group-vertical" style="margin:5px 0 10px 0;">
    <button type="button" class="btn btn-primary">Apple</button>
    <button type="button" class="btn btn-primary">Samsung</button>
    <button type="button" class="btn btn-primary">Sony</button>
</di>>
end.

Группы кнопок рядом друг с другом

По умолчанию группы кнопок отображаются «в ряд», то есть при наличии нескольких групп они располагаются рядом друг с другом:

Пример

<div class="btn-group" style="margin:2px 0 5px 0;">
    <button type="button" class="btn btn-primary">Apple</button>
    <button type="button" class="btn btn-primary">Samsung</button>
    <button type="button" class="btn btn-primary">Sony</button>
</div>
<div class="btn-group" style="margin:2px 0 5px 0;">
    <button type="button" class="btn btn-primary">BMW</button>
    <button type="button" class="btn btn-primary">Mercedes</button>
    <utton type="button" class="btn btn-primary">Volvo</button>
</div>
end.

Сгруппированные кнопки с выпадающим списком

Пример

<div class="container mt-3">
    <2>Nesting Button Groups</h2>
    <p>Nest button groups to create dropdown menus:</p>
    <div class="btn-group">
        <button type="button" class="btn btn-primary">Apple</button>
        <button type="button" class="btn btn-primary">Samsung</button>
        <div class="btn-group">
            <button type="button" class="btn btn-primary dropdown-toggle" data-bs-toggle="dropdown">Sony</button>
            <ul class="dropdown-menu">
                <li><a class="dropdown-item" href="#">Tablet</a></li>
                <li><a class="dropdown-item" href="#">Smartphone</a></li>
            </ul>
        </div>
    </div>
</div>

Nesting Button Groups

Nest button groups to create dropdown menus:

  • Tablet
  • Smartphone
end.
